# go-template

Template repository for Go project

## Build

Pointing to the root directory, run:

```bash
make
```

The compiled executable can be found in the `build/` directory.
Then, to run it:

```bash
./build/coolapp
```

Here, `coolapp` is the value of the `APPNAME` variable in the Makefile.

## Test

```bash
make test
```
